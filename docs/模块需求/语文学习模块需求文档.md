# 语文学习模块需求文档
## 一、模块概述
面向6-12岁小学生的语文学习功能模块，覆盖小学1-6年级语文核心知识点，提供趣味化学习方式，帮助学生提升汉字、古诗词、阅读理解等核心语文能力。模块设计符合儿童认知特点，界面活泼友好，操作简单直观。
## 二、核心功能需求
### 1. 古诗词背诵功能
| 功能点 | 详细要求 |
|--------|----------|
| 布局方式 | 采用Tab切换布局，三个功能模块（古诗词背诵、汉字听写、阅读理解）通过顶部Tab栏切换<br>技术实现采用Vue动态组件加载方式，三个功能模块独立拆分为子组件，Tab切换时按需加载对应组件，便于后续扩展维护 |
| 布局方式 | 古诗词背诵模块采用3:7左右两栏布局，左侧30%宽度为诗词列表，右侧70%宽度为诗词详情 |
| 诗词列表 | 左侧列表固定高度，内容过多时内部独立纵向滚动，不影响其他区域布局 |
| 诗词详情 | 右侧详情固定高度，内容过多时内部独立纵向滚动，顶部场景图高度为96px（h-24） |
| 内容展示 | 每首诗词包含场景配图、标题、朝代、作者、正文内容五部分 |
| 拼音要求 | 标题、朝代、作者、正文内容均需逐字对应拼音标注，准确标注声调，与统编版语文课本内容完全一致，格式为拼音在上、汉字在下，居中对齐 |
| 场景配图 | 每首诗词配套对应的场景图片，与语文课本插画风格一致，增强学习趣味性 |
| 年级匹配 | 诗词按照1-6年级进行分类，用户可选择对应年级内容学习 |
| 背诵辅助 | 支持点击播放按钮播放标准朗诵音频，优先使用豆包自然女声音源，无预存音频时自动降级为浏览器语音合成，优先选择微软晓晓等高自然度女声；<br>朗诵参数标准：语速调整为适合儿童的0.60倍，音量为1.2，音调为1.1，发音清晰柔和，符合小学教师带读节奏；<br>朗诵文本规则：标题后加停顿句号、朝代直接朗读、作者后加停顿句号，每句正文后加停顿句号；<br>支持完整的播放/暂停交互逻辑：<br>1. 暂停后继续播放时，高亮和语音从暂停位置无缝衔接<br>2. 播放完成后自动重置到初始状态，可再次点击播放<br>3. 切换诗词时自动停止当前播放并重置状态；播放时逐字同步高亮，智能匹配标点停顿节奏，当前正在朗读的文字和拼音自动变为醒目的橙色（#FF7F00）并放大，同时文字右上方显示蹦蹦跳跳的卡通小狗动画，增强跟读趣味性 |
| 背诵检测 | 提供挖空背诵模式，随机隐藏部分诗句，用户填写检测掌握情况 |
| 进度记录 | 记录每首诗词的学习次数、掌握程度（未学习/学习中/已掌握） |
### 2. 汉字听写功能
| 功能点 | 详细要求 |
|--------|----------|
| 汉字库 | 包含小学阶段要求掌握的3500个常用汉字，按年级和难度分级 |
| 听写模式 | 1. 看拼音写汉字 2. 听读音写汉字 3. 看汉字写拼音 |
| 发音功能 | 点击汉字播放标准普通话发音，支持慢速播放 |
| 笔画演示 | 支持汉字动态笔画书写演示，展示正确笔顺 |
| 组词扩展 | 每个汉字配套3-5个常用组词，辅助理解记忆 |
| 错题记录 | 写错的汉字自动加入错字本，可反复练习 |
### 3. 阅读理解练习
| 功能点 | 详细要求 |
|--------|----------|
| 文章分级 | 阅读材料按1-6年级难度分级，内容涵盖寓言、童话、记叙文、说明文等 |
| 题型设计 | 包含选择题、填空题、简答题等多种题型，符合考试题型要求 |
| 答题反馈 | 答题后即时反馈对错，提供详细的答案解析 |
| 生词标注 | 文章中生僻字词支持点击查看拼音和释义 |
| 朗读功能 | 支持全文朗读和逐句朗读，帮助提升语感 |
| 能力评估 | 根据答题正确率自动评估阅读理解能力水平，推荐合适难度的文章 |
### 4. 学习进度统计
- 展示今日语文学习总时长、完成练习数量
- 各知识点掌握率统计（汉字、古诗词、阅读理解）
- 连续学习天数统计
- 正确率变化趋势展示
## 三、UI设计要求
1. **整体风格**：延续平台统一的卡通活泼风格，以红色系为主色调（#E53E3E）
2. **导航规范**：
   - 所有二级及以下页面统一使用面包屑导航，格式为：首页 > 父级页面 > 当前页面
   - 面包屑中的首页和父级页面支持点击跳转，使用灰色字体，hover状态高亮显示主色#3399FF
   - 面包屑当前页面文字使用对应科目模块主色调（语文模块为红色#E53E3E），不可点击
   - 页面无需重复显示面包屑中已有的模块标题，避免冗余展示
3. **字体要求**：默认字号14px，诗词正文和汉字练习使用18-24px大号字体，拼音标注使用12-14px字体
4. **布局要求**：
   - 功能卡片式布局，每个功能模块独立成卡
   - 操作按钮尺寸≥48x48px，适合儿童点击
   - 内容区域留白充足，避免信息过载
5. **交互反馈**：
   - 点击按钮有放大+阴影变化效果
   - 答题正确显示绿色动画+星星奖励
   - 答题错误显示红色提示+鼓励性文字
## 四、数据结构规范
### 古诗词数据结构
```typescript
interface Poem {
  id: string;
  titleChars: string[];   // 标题逐字拆分数组
  titlePinyin: string[];  // 标题拼音数组，与汉字逐字对应
  dynastyChars: string[]; // 朝代逐字拆分数组
  dynastyPinyin: string[];// 朝代拼音数组
  authorChars: string[];  // 作者逐字拆分数组
  authorPinyin: string[]; // 作者拼音数组
  grade: number;          // 对应年级 1-6
  contentLines: string[][];  // 诗句二维数组，每行诗句为一个数组，每个元素为单个汉字
  pinyinLines: string[][];   // 拼音二维数组，与contentLines逐字对应
  audioUrl?: string;      // 朗诵音频地址
  mastery: number;        // 掌握程度 0=未学习 1=学习中 2=已掌握
}
```
### 汉字数据结构
```typescript
interface ChineseCharacter {
  id: string;
  character: string;    // 汉字
  pinyin: string;       // 拼音
  strokes: number;      // 笔画数
  radical: string;      // 部首
  grade: number;        // 对应年级
  words: string[];      // 常用组词
  strokeOrder: string[];// 笔顺数组
  audioUrl?: string;    // 发音音频
}
```
## 五、非功能需求
1. **性能要求**：页面加载时间≤1.5s，音频播放响应时间≤500ms
2. **兼容性**：支持Chrome、Edge、Safari浏览器，适配平板设备横屏显示
3. **无障碍要求**：所有可点击元素有清晰的视觉反馈，支持键盘操作
4. **数据存储**：学习记录和进度本地存储，支持云端同步
