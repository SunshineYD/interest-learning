# 6-12岁儿童学习网站需求文档
## 一、项目概述
面向6-12岁学龄儿童的在线学习平台，覆盖语文、数学、英语核心科目学习，配套学习记录、打卡、统计分析、知识点掌握图谱等功能，打造趣味化、可视化的学习体验。
## 二、核心功能模块
### 1. 科目学习模块
| 科目 | 核心内容 |
|------|----------|
| 语文 | 汉字听写、古诗词背诵(需要标注拼音，与诗词内容对应，与课本内容一致)、阅读理解练习 |
| 数学 | 口算练习、应用题训练、公式卡片学习、奥数启蒙 |
| 英语 | 单词记忆、听力训练、口语跟读、绘本阅读 |
> 每个科目练习支持难度分级，匹配1-6年级不同学习阶段
### 2. 学习记录模块
- 每次学习的开始/结束时间、学习时长记录
- 完成的练习内容、正确率、错题记录
- 学习内容自动关联对应知识点
### 3. 打卡系统模块
- 每日学习打卡功能，支持打卡后获得积分/勋章奖励，积分和勋章数量根据学习内容难度和正确率动态调整，难度越高，正确率越高，奖励也越高。
- 积分和勋章均可兑换奖励物品，如游戏道具、玩具、图书等。
- 连续打卡天数统计，断签提醒
- 打卡日历可视化展示，直观查看每月打卡情况
### 4. 统计分析模块
- 近7天学习统计：总学习时长、各科目占比、完成练习数量
- 近30天学习统计：学习趋势图、正确率变化曲线
- 错题汇总统计：高频错题分类，薄弱知识点标注
### 5. 知识点掌握图谱模块
- 按科目构建知识树结构，可视化展示所有知识点
- 已掌握知识点标注为绿色，薄弱知识点标注为红色，未学习知识点标注为灰色
- 支持点击知识点查看对应学习内容和练习入口
### 6. 儿童友好功能
- 界面色彩明亮活泼，卡通元素设计
- 字体放大显示，适合儿童阅读
- 学习时长提醒，防止用眼过度
- 家长后台：可查看孩子学习情况，设置学习时长限制
## 三、技术选型建议
| 层级 | 技术栈 |
|------|--------|
| 前端 | Vue.js + TypeScript + Tailwind CSS + shadcn/ui |
| 状态管理 | Pinia + Vue Query |
| 后端 | Next.js API Routes |
| 数据库 | SQLite（轻量易部署）/ PostgreSQL |
| ORM | Prisma |
| 图标 | Lucide 卡通风格图标 |
## 四、UI设计要求
1. 整体风格：卡通、活泼、明亮，使用高饱和度低明度的柔和配色方案，如#3399FF、#FF9933等，网站风格各个页面保持一致。
2. 交互设计：大按钮、简单操作流程，减少复杂操作步骤
3. 反馈设计：操作成功/失败使用动画+音效提示（可选）
4. 响应式：支持PC端和平板设备访问
5. 统一处理相同类型按钮点击事件，如学习按钮、打卡按钮、统计分析按钮等，点击后触发对应操作。
6. 点击后显示对应操作结果，如学习按钮点击后显示学习内容，打卡按钮点击后显示打卡结果，统计分析按钮点击后显示统计分析结果等。
7. 界面布局：清晰、有序，避免信息过载。
8. 字体大小：14px，字体颜色：#333，背景颜色：#F5F5F5
## 五、非功能需求
1. 性能：页面加载时间<2s，操作响应时间<500ms
2. 兼容性：支持Chrome、Edge、Safari主流浏览器
3. 数据安全：用户学习数据本地存储+云端备份可选
4. 扩展性：支持后续新增科目（科学、美术等）和功能模块
## 六、后续开发优先级
1. 基础框架搭建 + 科目学习核心功能
2. 学习记录 + 打卡系统
3. 统计分析 + 知识点图谱
4. 家长后台 + 趣味激励系统
