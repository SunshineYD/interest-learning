
# 变更摘要：语文模块已掌握功能完善及UI优化
**时间戳**：2026-04-13 14:00
**变更类型**：功能开发 + UI优化
**影响范围**：语文模块古诗词背诵功能
## 核心变更内容
1. **功能开发**：实现"我会背了"掌握状态标记功能
   - 诗词列表项显示"已掌握"红色盖章标签
   - 诗词详情标题右上方显示透明"已掌握"盖章
   - 支持切换掌握/未掌握状态
2. **列表样式优化**：
   - 未选中状态默认使用浅灰色背景 `bg-gray-50`
   - hover 状态加深为 `bg-gray-100`
   - 列表项固定最小高度 `min-h-[60px]`，避免掌握状态切换时高度变化
3. **盖章样式优化**：
   - 列表盖章：红色渐变背景，旋转-5度，仿盖章效果
   - 详情盖章：半透明红色渐变背景，旋转20度，更像真实印章
   - 位置调整：标题右侧外25px，不覆盖标题内容
4. **规范文档更新**：
   - 在全局功能交互规范中新增开发流程规范
   - 明确开发服务器启动规则，禁止自动启动

## 验证要点
✅ 点击"我会背了"按钮，列表和详情同时显示"已掌握"盖章
✅ 点击"取消掌握"，恢复到未掌握状态
✅ 诗词列表项有默认浅灰色背景，hover时加深
✅ 诗词标题不折行，始终横向完整展示
✅ 透明盖章位于标题右上方，不覆盖标题
✅ 列表项高度固定，掌握状态切换时无高度变化
